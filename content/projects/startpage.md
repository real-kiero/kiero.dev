A personal browser start page built to replace the default new-tab experience. This project holds a special place for me as it was both a fun experiment and the first *"real"* repository I shared on GitHub. To my surprise, it was very well received and even picked up a few forks after I shared it on [r/startpages](https://www.reddit.com/r/startpages/comments/i2jd37/clean_startpage_dark/).
